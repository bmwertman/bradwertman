Know Thy Selfie

Inspired by regional selfie characteristics as demonstrated by SelfieCity. Know Thy Selfie, built on Rails & Backbone, uses client side Javascript to grab a device's camera. A taken photo is then animated from a Polaroid camera to appear as if it were 'printed' & 'developed'. It is then stored on Amazon S3, analyzed by a facial recognition API & has a corresponding "Sharpie" comment appended before being displayed. Checkout the code on Github!

Balderdash

A real time multi player game where players are presented with an obscure word. Players then create fraudulent definitions, we mix in the correct one, and everyone votes on which is the real one. Ajax requests are used to continually check the game state for all player views and synchronize the views once all definitions or guesses have been submitted. Check out the code on Github!
